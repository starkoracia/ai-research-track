## 3. Main idea

The main idea I understood is that a Jupyter notebook is made of cells. Code cells execute Python code, and Markdown
cells contain explanations, headings, lists, and notes. JupyterLab is a wider workspace where I can work with notebooks,
files, terminals, and outputs. I have also seen that notebooks can display results such as text output and graphs
directly under the code cell.

## 4. Key takeaways

- Notebook: a document that can contain code, text, outputs, and graphs.
- Code cell: a block where Python code can be executed.
- Markdown cell: a block where I can write formatted explanations.
- Kernel: the Python process that runs the notebook code.
- Output: the visible result of running a code cell.
- Plot / graph: a visual output created from data.
- JupyterLab: a browser-based workspace for notebooks, files, terminals, and documents.
