# AI Research Track

This repository contains my learning artifacts for the first part of the AI Research and Engineering Track.

## Purpose

The purpose of this project is to build a reproducible research workspace for studying machine learning and AI.

## Structure

- `notebooks/` — Jupyter notebooks for experiments
- `src/` — reusable Python code
- `data/` — data documentation and local datasets
- `reports/` — short reports and reflections
- `notes/` — research notes
- `references/` — source lists and bibliography notes

## Week 01 Goal

The goal of Week 01 is to create a clean project structure, initialize Git, write the first NumPy notebook, and prepare the first research notes.

## How to run

Install dependencies:

```pip install -r requirements.txt```
